# POD Condesa — Web Build Prompt You are building the official website for **POD Condesa**, a multidisciplinary space in Mexico City's Condesa neighborhood that fuses an art gallery, creative coworking, café, rooftop, and wellness hub. The site is the digital extension of the brand's soft-opening identity: editorial, sensorial, and unmistakably contemporary. Build a single-page, scroll-driven website. Use **HTML + CSS + vanilla JS** (or Next.js / Astro / React if explicitly requested). No frameworks beyond what's needed. Production-grade. No placeholder lorem ipsum — use the real copy below. --- ## Brand System (non-negotiable) ### Colors ``` --blue: #1E1EE6 /* Klein blue, primary accent */ --cream: #EFEFE0 /* warm off-white, primary background */ --black: #0A0A0A /* near-black, ink */ --ink: #111111 /* text default */ ``` Dominant palette is **cream + ink**. Klein blue is used as a sharp accent (waves, handwritten flourishes, hover states, one full-bleed section). Black sections appear at night-themed moments (rooftop / HOG party). Never use gray gradients, never use white pure (#FFF). ### Typography - **Display / UI**: `Poppins` — weights 400, 500, 600, 700, 800. Use ExtraBold/800 for headlines with tight tracking (`letter-spacing: -0.02em`). - **Handwritten accent**: `Caveat` Bold — only for short emotional phrases ("your new favorite", "friends & family", "see you soon"), always in Klein blue, slight rotation (-3° to +3°). - **Mono-style micro-labels**: Poppins SemiBold, uppercase, tracking `0.25em`, size 10–12px. Used for tags like "POD · CONDESA", "NUEVO LEÓN 108", "EST · 2025". ### Visual Motifs - **Blue squiggle waves** — the brand's signature. Hand-drawn SVG paths with thick rounded strokes (40–80px), bezier curves, slightly imperfect. They cross sections diagonally, never decorative-only — they bridge content. - **Numbered captions** like `(001)`, `(002)`, `(003)` in blue, used as editorial section markers. - **Layered grain / noise** subtle overlay on cream backgrounds (`background-blend-mode: multiply` with a low-opacity noise SVG). - **Polaroid-style photo treatments** for the gallery section: white border, slight rotation, soft shadow. - **Black-and-white film photography** dominates. Color photos only for warm interior shots (golden hour, candlelight). ### Logo - **Wordmark**: lowercase `pod` in Poppins ExtraBold, very tight kerning. - **Symbol**: 4-pointed star / diamond petal mark (the one from the brand manual). Provide as inline SVG. - **Tagline lockup**: `pod` + small caps "PURPOSE DRIVEN HUB" stacked. --- ## Page Structure ### 0. Loader / Intro (2 seconds max) Cream background. All in micro-label style. --- ## Interaction & Motion - **Scroll-triggered reveals**: each section's headline animates with a mask-reveal (text wipes up from below), staggered 80ms per line. Use IntersectionObserver, not heavy libs. - **Cursor**: custom on desktop — a small Klein blue dot that grows into a ring on interactive elements. Hide on touch. - **Wave SVGs**: animate `stroke-dashoffset` from full length to 0 when they enter viewport, 1.2s ease-out. - **Marquee** in hero: pure CSS `@keyframes` translate, pauses on hover. - **Hover on buttons / cards**: 200ms ease, never a jarring snap. Color inversions feel like ink dipping into paper. - **No parallax scroll-jacking**. No GSAP unless you can justify it. Minimal, intentional motion. --- ## Responsive Rules - **Desktop ≥1200px**: full editorial asymmetry, 12-col grid. - **Tablet 768–1199px**: collapse asymmetric layouts to 6-col, keep type scale aggressive. - **Mobile <768px**: single column, but **keep the brand voice loud** — don't soften display sizes too much. Clamp() on headline sizes. Stories-style snap-scroll for the artist row. - Touch targets ≥44px. Cursor effect disabled. --- ## Accessibility & Performance - Semantic HTML5 (`

Inline SVGs for the logo mark and all waves — never raster these. --- ## Anti-Patterns (what NOT to do) - ❌ No purple-to-blue gradients. - ❌ No glassmorphism, no neumorphism, no generic SaaS hero pattern. - ❌ No Inter, no Roboto, no Arial as fallback display. - ❌ No "Get Started Free" CTA energy. This is a cultural space, not a startup. - ❌ No emojis in copy. The handwritten Caveat phrases carry the warmth. - ❌ Don't center-align everything. Editorial sites breathe through asymmetry. - ❌ No stock photos of generic "diverse team meeting." Use only the moody, candlelit, film-grain aesthetic of the brand manual. --- ## Reference Voice The site should feel like **Acne Studios meets a Berlin gallery booklet meets a Mexico City wine bar menu**. Quiet luxury, but with a handwritten human edge. Editorial, not corporate. Sensorial, not informational. Final test: if you can't imagine the site being printed as a zine and handed out at the opening, you've over-designed it. Build it.
